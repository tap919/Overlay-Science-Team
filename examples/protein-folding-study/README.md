# Example: Protein Folding Study

This example demonstrates how to use the Overlay Science Lab VSCodium extension to conduct a protein folding research study.

## Overview

- **Study Name**: Protein Folding Analysis 2026
- **Objective**: Analyze protein folding pathways using quantum chaos theory
- **Tools Used**: GLCCE, Digital Lab, PubChemPy, Biopython

## Quick Start

### 1. Open in VSCodium

```bash
cd Overlay-Science-Team/examples/protein-folding-study
codium .
```

### 2. Install Dependencies

```bash
pip install pubchempy biopython scikit-bio pandas matplotlib
```

### 3. Run Study

```python
python study_setup.py
```

After pipeline completion:

workspace/deliverables/protein-folding-2026/
